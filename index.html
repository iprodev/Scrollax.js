Parallax has never been easier.

Add some simple data attributes to your markup, run $.Scrollax();.
That's all you need to get started.

Now get this thing on GitHub and spread the word, it's open source!

All examples (except the 'Particles' one) are using a calling via jQuery proxy, because some people that are looking only at examples think that jQuery is a separate language and can't handle raw JavaScript. This'll hopefully help with lowering the "help me make it work" issue reports.

More advanced users can surely read the documentation and discover all the power & cool stuff that new Scrollax() provides :)
